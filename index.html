hello,Hexo

emm,my name is shuijianbaozi

  • 介绍
    • hexo:基于node.js的,轻量级的,高性价比的,静态博客框架
    • github:全球最大的,同性交友网站

  • 操作步骤

    • 安装Node.js和配置好Node.js环境，打开cmd命令行测试
      • node -v
      • npm -v
    • 安装git和配置git环境,打开cmd命令测试
      • git –version
    • github上创建一个新的项目(遵守格式:账户名.github.io)
      • 访问:https://shuijianbaozi.github.io
    • 创建文件夹(该文件夹为博客放置的文件夹)
      • 在f盘创建blog文件夹
      • dos窗口进入blog文件夹
        • cd f:/blog
    • 安装 Hexo
      • npm install hexo -g #开始安装hexo
      • hexo -v #检查hexo是否安装成功
      • hexo init #使用Hexo初始化文件夹
      • npm install #安装所需要的组件
      • 首次体验Hexo
        • hexo g
        • hexo s #开启服务器
          • hexo server -p 4001 #如果端口被占用
        • 浏览器测试:http://localhost:4000/
        • ctrl+c #停止服务器
    • 将Hexo与Github page联系起来(设置git的username和useremail)
      • git和blog文件夹关联
        • 常用方法
          • 进入F:\blog
          • 点击Git Base Here
          • git config –global user.name “shuijianbaozi” #配置全局的name(github账号name)
          • git config –global user.email “1649258746@qq.com“ #配置全局的email(github账号email)
          • cd ~/.ssh
          • ls
          • ssh-keygen -t rsa -C “1649258746@qq.com“ #生成密钥(默认文件路径C:\Users\Administrator.ssh)
          • 连续点击三次回车 设置的密码为空
          • eval “$(ssh-agent -s)” #添加密钥到ssh-agent
          • ssh-add ~/.ssh/id_rsa #添加生成的SSH key到ssh-agent
        • windows7
          • 开始-Git Bash #打开窗口
          • cd f/blog #进入blog文件
          • make ~/.ssh #创建.ssh文件(默认文件路径C:\Users\Administrator.ssh)
          • git config –global user.name “shuijianbaozi” #配置全局的name(github账号name)
          • git config –global user.email “1649258746@qq.com“ #配置全局的email(github账号email)
          • ssh-keygen -t rsa -C “1649258746@qq.com“ #生成key
          • 连续点击三次回车 设置的密码为空
          • eval “$(ssh-agent -s)” #添加密钥到ssh-agent
          • ssh-add ~/.ssh/id_rsa #添加生成的SSH key到ssh-agent
    • 在github中添加ssh
      • 登陆Github-头像-settings-Personal settings-SSH and GPG keys-new ssh key-将.ssh文件中的id_rsa.pub内容复制
      • 按钮(Add SSH key)-输入github的账户密码
      • git窗口输入ssh -T git@github.com #测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了

    • 配置Deployment文件,在文件夹中,找到_config.yml文件,修改repo的值(在末尾),注意:下边type,repository,branch后边都有空格!!

    • 新建一篇博客

      • f:\blog>hexo new post “hello,Hexo” #这时候在文件夹blog/source/_posts目录下将会看到已经创建的文件
      • npm install hexo-deployer-git –save #生成和部署文章之前,安装一个扩展(只需要下载一遍即可)
      • hexo d -g #编辑好的文章生成部署
      • github网站上刷新一下即可看到几秒前用git更新的内容
      • 部署成功后访问你的地址:http://用户名.github.io
        • https://shuijianbaozi.github.io/index.html 也可以访问
        • 如果有乱码问题,直接在编辑器哪里修改文件为utf-8的格式即可
